# Shri Naina Enterprises

A modern, clean, responsive business website for **Shri Naina Enterprises** — a retailer and supplier of premium mattresses and pillows, powered by **Spenta Mattress**.

**Tagline:** *Comfort You Can Trust*

## Tech Stack

- **React 18** (Vite)
- **Tailwind CSS**
- **Framer Motion** (smooth animations)

## Features

- Sticky navbar with scroll-aware blur
- Hero section with animated gradient background
- About section
- Products showcase (Mattresses + Pillows) with "Powered by Spenta Mattress" branding
- Why Choose Us (Premium Quality, Affordable Pricing, Trusted Brand, Customer Satisfaction)
- Contact section with click-to-call, click-to-mail, and UI contact form
- Location section with "View Location" button linking to the map
- Fully responsive footer
- Soft blue/purple/white theme, rounded cards, smooth scroll animations

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
